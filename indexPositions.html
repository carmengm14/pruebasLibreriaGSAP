Static

Este párrafo tiene la posición predeterminada.

Relative

Este párrafo se desplaza hacia arriba y hacia la derecha en relación con su posición original.

Absolute

Este párrafo se posiciona en relación con el ancestro no estático más cercano.

Fixed

Este párrafo se posiciona en relación con la ventana del navegador y se queda fijo en la misma posición incluso si el usuario hace scroll hacia arriba o hacia abajo.

Sticky

Este párrafo se comporta como relative hasta que el usuario hace scroll hacia él, en ese momento se convierte en fixed.
